# DataBase Commands
1. View all databases  
`show dbs`

1. Create a new or switch databases   
`use name_of_database`

1. View all databases  
`db`

1. View all databases  
`db.dropDatabase()`


# Collection Commands
1. Show Collections  
`show collections`

1. Create a collection named 'coll_name'   
`db.createCollection('coll_name')`

1. Drop a collection named 'coll_name'  
`db.coll_name.drop()`
